Page structure

Standalone page — no AppLayout wrapper. Uses the aero-bg background class so both versions sit on the Frutiger Aero canvas. No sidebar, no top nav (it's a prototype, not a real page).

Layout

Version A — Current
Glassmorphism skeu-cards
No sparklines
Decorative bubbles (shockwaves)
No overwhelm score
Balance: 3 mode buttons
Version B — Tufte
Solid bg-white cards
SVG sparklines (colour = trend)
No bubbles behind cards
Overwhelm score 0.32
Balance: draggable slider

Side-by-side on ≥768px, stacked on <768px. Both columns scroll together.

Sparklines (Version B only)

Raw inline SVG polylines — no recharts (not installed). 4-point data rendered as a small 80×24px chart beneath the number. Colour logic:

↑ Improving → green stroke
↓ Worsening → orange stroke
→ Stable → grey stroke

Mock data as specified: [28,31,34,36] total, [25,28,30,33] completed, [2.1,1.8,1.3,1.5] hours, [3,1,2,0] due soon.
